# Bassa's Dotfiles

## Install

Before install sign in to iCloud and Mac App Store

Then

```bash
bash -c "$(wget -O - https://git.io/dfboot)"
```

Or

```bash
bash -c "$(curl -L https://git.io/dfboot)"
```

## Credits

With lots of cool ideas taken from [Github does dotfiles](https://dotfiles.github.io/)

## Wiki

To fix short URL:

```bash
curl -i https://git.io -F "url=https://raw.githubusercontent.com/norville/dotfiles/master/bin/dfboot" -F "code=dfboot"
```
