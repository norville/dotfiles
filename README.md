# Bassa's Dotfiles

## Why

To quickly restore my computer environment: apps, tools, frameworks and preferences.

## How

With lots of cool ideas taken from [Github does dotfiles](https://dotfiles.github.io/)

## Install

```bash
xcode-select --install && curl -L https://raw.githubusercontent.com/norville/dotfiles/bare/bin/dfboot | bash
```
Or
```bash
xcode-select --install && wget -O - https://raw.githubusercontent.com/norville/dotfiles/bare/bin/dfboot | bash
```
